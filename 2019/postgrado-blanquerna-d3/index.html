Visualizando elecciones

Una introducción práctica a R y D3

Diploma d'Especialització Universitària en Periodisme de Dades i Visualització, Blanquerna

Martín González Gómez

Barcelona, 3 y 4 de mayo, 2019

En este taller preparemos una visualización de datos usando R y D3, paso a paso. Si nos da tiempo daremos el salto a tooltips, transiciones, y diseño móvil. Para poder inspeccionar y tocar esta web hay que instalar un servidor local. Comprobamos que está node instalado. Si no, lo instalamos.

Ahora, podemos descargarla desde la carpeta compartida del postgrado, en SLIDES/D3-Martin.

Ahora, hay que instalar el servidor. Para eso, podemos seguir estas instrucciones que utilizan Python.

Si instalaste node en el taller, puedes navegar de nuevo a la carpeta usando cd, por ejemplo cd Descargas/posgrau-d3, y luego ejecutar http-server e ir a localhost:8080.
